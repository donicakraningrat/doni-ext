Syarat Pendukung:
1. Terinstall Node.js di local machine

Installasi:
2. Buka Chrome 
    - tempelkan di address bar URI: "chrome://extensions/"
    - nyalahkan "Developer Mode"
    - klik tombol "Load unpacked" dan cari lokasi FOLDER dari chrome extensions.
    - selesai.

Installasi API:
1. Jalankan terminal dan arahkan sesuai lokasi applikasi.
2. ubah file .env di dalam folder sesuai konfigurasi yang di butuhkan:
    - port untuk menentukan port api 
3. Jalankan di terminal: npm Install
4. Jalankan di terminal: node index.js
    - hingga muncul pesan Server is running on port ????

Menghidupkan API:
1. buka terminal arahkan lokasi chrome extensions
2. Jalankan di terminal: node index.js
    - hingga muncul pesan Server is running on port ????
